Twitter Bot

I wrote a Twitter bot using Javascript to tweet the top 5 rankings in the F1 Driver's Championship evry week. The data was scraped from ESPN using the Puppeteer library, and the bot connected to Twitter through the Twitter API. It was deployed using Heroku Cloud Application Platform.
