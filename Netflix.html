Netflix Original Films Analysis

In this project, I analyzed data from Netflix Original Films & IMDB Scores. The data set contains info about Original films produced by Netflix up until June 1st, 2021. The data and code used in this project can be found here.

The objective I had in mind when starting this project was to create an interactive dashboard to display the most popular release times as well as which genres Netflix performs the best in. On the "Netflix Originals" dashboard, the release day, month, and year are displayed. On the "Distribution of IMDB Scores" dashboard, the score distribution for all movies is displayed as well as the correlation between film runtime and score. Both of these dashboards can be filtered to display all info based on the top 5 genres.

The following charts shows which languages were used the most, and the 10 movies that recieved the best and worst scores from IMDB.

The following Python code calculates the correlation coefficients between Runtime, Year, and IMDB Score.

import pandas as pd

#read Runtime, Year,  and IMDB Scores from csv
fields = ['Runtime','Year', 'IMDB Score']
data = pd.read_csv(r'D:\\Projects\\NetflixOriginalFilms\\NetflixOriginalsCSV.csv', usecols = fields, encoding = 'latin1')
								
#Finding correlation between Runtime/Year/IMDB Scores
corr = data.loc[:,fields].corr()
							
print(corr)

Running this code calculates a correlation coefficient of -0.041872 between runtime and score, and -0.140982 between year released and score. This means that the runtime of a film and year released have almost no impact on its IMDB score.

Conclusion

  • The length and release date of a film has basically no impact on its IMDB score.
  • Documentaries, Dramas, and Comedies are the top 3 genres at Netflix, making up almost half of all films they produce.
  • Most films are in English, Hindi, or Spanish.
  • Most films have an IMDB rating between 5.0 and 7.5.
  • Netflix have continued to produce more films each successive year. (2021 wasn't over at the time this data was created.)
  • Netflix's best performing genre was their documentaries
